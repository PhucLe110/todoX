# 📝 TodoX

> **Dự án học tập Fullstack MERN**  
> Xây dựng ứng dụng quản lý công việc (To-do App) với **React + Node.js + MongoDB + Tailwind CSS v4 + Shadcn**  
> Đồng thời thực hành **triển khai (deploy) trang web** lên môi trường thực tế.

---

## 🚀 Công nghệ sử dụng

- **Frontend:** React + Tailwind CSS v4 + Shadcn UI
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Triển khai:** Render

---

## 🎯 Mục tiêu dự án

- Học cách làm dự án **Fullstack MERN** hoàn chỉnh.
- Nắm vững cách **kết nối frontend & backend**.
- Quản lý dữ liệu với **MongoDB**.
- Tích hợp UI hiện đại với **Tailwind v4 + Shadcn**.
- Thực hành **deploy** website.
